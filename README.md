# stn-docker
Scala-TypeScript-Nodejs docker

- [x] Scala & sbt
- [x] Nodejs & npm
- [x] TypeScript

## Quick Start
### Start Docker
```shell
docker run -it --privileged stn:v1
```

The `--privileged` option is compulsory for the initialization.

### Get Your Project
```shell
git clone ...
```

- [ ] Support SSH

### Install other dependencies
```shell
yay -S ocaml
```
